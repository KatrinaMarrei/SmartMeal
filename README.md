# SmartMeal

SmartMeal — учебный веб-сервис для планирования питания на неделю.

## Технологии

- C#
- ASP.NET Core MVC
- Razor Views
- Entity Framework Core
- SQL Server LocalDB
- HTML/CSS
- Git/GitHub

## Ограничения проекта

- Приложение запускается локально.
- Интерфейс реализуется через Razor Views.
- JavaScript не используется для логики приложения.
- Данные хранятся в локальной базе SQL Server LocalDB.

## Основные модули

- Управление пользователями
- Каталог блюд
- План питания на неделю
- Профиль пользователя
- Отчётность по калориям и БЖУ
